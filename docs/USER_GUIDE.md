# SuperLand User Guide

SuperLand has two main areas: the customer website and the admin dashboard.

## Customer Website

Open:

```text
http://localhost/SuperLand/
```

Main customer features:

- Register a customer account
- Log in as a customer
- Browse products
- Browse products by category
- Search products
- Add products to cart
- Update cart quantity
- Checkout
- Update customer profile

Demo customer account:

```text
Email: user@superland.com
Password: user123
```

## Admin Dashboard

Open:

```text
http://localhost/SuperLand/admin/AdminDashboard.php
```

Main admin features:

- Admin login
- View admin dashboard
- Add, edit, and delete products
- Add, edit, and delete categories
- Add, edit, and delete admin users

Demo admin account:

```text
Email: admin@superland.com
Password: admin123
```

## Important Notes

- Import the database before using the system.
- Do not rename folders unless the related code paths are updated.
- Product and category images are stored inside the project assets folders.
- Uploaded user images are stored inside the uploads folders.
